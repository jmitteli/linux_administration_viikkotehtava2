# linux_administration_viikkotehtava2
Asenna ns. LEMP-stack CSC:n cPouta palveluun, jossa on käynnissä Nginx-serveri pyörittäen staattista HTML-sivua HTTP-portissa. Muista avata portti kaikille, jotta vertaisarvioija pääsee arvioimaan työsi. Pidä huoli, että service pysyy käynnissä.

HUOM! Sinulla täytyy pyöriä viikkotehtävä 1 vielä viikon ajan, joten joudut luomaan uuden virtuaalikoneen tähän tehtävään. Tai sitten voit käyttää samaa konetta mutta joudut konfiguroimaan Nginx-serverin kuuntelemaan eri porttia kuin Apache-serveri. Tässä tapauksessa URL-osoitteessa täytyy näkyä ko. portti.

Oheisen ohjeen lisäksi:

Käytä Githubia versionhallintaan (tämä helpottaa seuraavia tehtäviä)

Personoi sivua.

Lisää sivulle HTML-elementti, joka näyttää kellonajan, pyydettynä SQL-serveriltä.

Älä lisää nimeäsi sivulle, jos haluat pysyä anonyymina vertaiselle. Palauta URL:it sivullesi ja Github-repoon (Pidä repo julkisena)
